Personal Website



This was one of the first websites I created and was an assignment for university. The aim of the assignment was to gain an understanding of html, css, javascript and implement an animation into the website. Each part of the website accessible through the navigation bar at the top, serves to meet an objective for the assignment. The home page holds a small introduction and the video page holds a video which can be played. The hobbies page discusses a couple of my hobbies. The portfolio page contains my portfolio from the time I created the website and finally the animation page holds the animation.

This website could have been improved by considering how it would appear on devices of different sizes and changing elements accordingly. It also would have benefitted from a more visually appealing interface such as centering the navbar and improving the layout of the elements on the home and hobbies pages. Despite this, creating this website proved effective at introducing me to html, css and javascript. From this baseline, I have then been able to build on my knowledge and create more complex web pages moving forwards.
