Original Character Website



This website was made as a part of a university assignment. The aim of the assignment was to create a web application containing a home page, 3D model pages for 3 3D models and an about page discussing how I made the website and models. User testing was also a requirement of the assignment. I created a web app that met the above specifications and was themed around an original character I created. The home page introduces people to the website and contains a small overview about each model and a find out more button to link them to the page. The model pages can also be accessed through the navigation bar at the top. These pages hold the 3D models and multiple ways in which they can be manipulated. A user can play animations and can switch between animations on models with multiple. They can also view the wireframe and rotate the model. Each model page utilises a single html page with a script to change the data depending on the model the user selected when switching to the page. Finally, the about page provides information about the design of the website and how the 3D models were implemented. Each image across the website is able to be clicked on to make them larger as well.

This site could have been improved through adding more controls to the 3D models and ways to manipulate them such as changing the background colour or allowing them to move around the scene. It could also be improved by carrying out more thorough testing to discover and fix any issues that may be present in the website.
