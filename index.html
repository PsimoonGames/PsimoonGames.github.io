Introduction

Welcome to my website! This portfolio highlights my favourite projects I have undertaken in web and game development. This is not a finite record of all the projects I have completed, but instead showcases the projects that taught me the most, challenged my skills and strengthened my abilities.

Below are three projects I am especially proud of - ones I feel deserve special attention. To see more, open the portfolio dropdown and choose a category. Enjoy exploring!

GravTrap

This project was made as my final year project at university. I created a game which bases itself around movement utilising mechanics such as sliding, dashing and wallrunning. It also has enemies for the player to defeat in the form of turrets and utilises gravity changing direction as a way to mix up gameplay.

PayApp

This web application was made using Django for the backend logic. The application features a registration system along with ways for users to send and request money, see previous transactions and see their balance. There are also administrator users which can create new administrators and see users and transactions.

Original Character Website

The aim of this website is to display and show 3D models. It contains a total of 3 3D models and various interactions such as showing its wireframes and playing animations. The website also contains an about page providing information about how the various parts of the website works and a homepage. The website also utilises bootstrap to lay out the pages, to improve the structure of the website and enable it to work on devices of different screen sizes.
