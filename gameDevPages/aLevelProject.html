A Level Project



This project was my first time trying game development. For this project I was guided to use the Monogame framework which uses the C# programming language. Previously I had been taught how to use python, however I had learnt no other programming language so this was also my first time using C#. I was incredibly ambitious with what I could achieve expecting to create a short story game featuring combat and puzzles along with a narrative. Being so inexperienced, this was an unrealistic goal so the final result differed greatly from my initial aim. However, I was still able to create a short game with a narrative and implement a basic puzzle which is used a couple of times.

Despite this, I was able to learn a lot from this project. The main takeaway was leaning C# as a programming language and growing comfortable in using it. Another main takeaway is that I needed to manage the scale of projects moving forwards. I gave myself too much to do in the time provided with this project, and as a result, some features were not able to be completed.
