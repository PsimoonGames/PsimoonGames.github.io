Game Design And Development Project



This project was made for a University module and required students to work as a group to create a game. The primary requirements for the game was that it had to be a 3D game and that it includes some kind of collectibles for the player to collect. As a group, we ended up deciding to create a movement based game, where the player would traverse levels and collect coins. Collecting all the coins would cause the player to win the game. The final game we created meets these requirements, however, the movement system could have been improved through better testing and allowing more time to make adjustments. This is because it ended up feeling very floaty and not as smooth and predictable as intended.

Throughout the duration of this project I got to learn how to use many features of Unity and get to grip with all the tools it offered. With it being a group project, I also learnt about working in a group and how to manage the different tasks that needed completing between the different members of the group. It was recommended that we use git to track progress and share our work. Before this project, I had never used git before, so this was another major tool I learnt how to use during this project.
