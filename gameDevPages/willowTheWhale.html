Willow The Whale



This game was created in a week as a part of Jame Gam 50, hosted on Itch.io. The theme for this game jam was Space Whales, and there was a requirement to include a special item. For this game jam it was a printed circuit board. The game I created was an adventure game about a space whale accidentally eating a printed circuit board which then guides the whale to a destination. Navigation is achieved through a procedurally generated map of different solar systems, each one containing a combat encounter for the player to complete before moving onto the next. Each combat encounter is turn based, where the player and enemies have a selection of moves they can use each turn. I would have liked to have been able to add minigames for each move the player makes to determine the amount of damage or healing they do, however, in the interest of making the game more balanced and optimised, I decided to leave that feature out of the game I submitted.

The main thing I learnt during the creation of this game was how to implement procedural generation. It's a feature that has historically interested me and being able to implement it in this game was a good start to learning about it. The way this game implements it is by using a point map, and then randomly moving the points a certain distance and direction. From this, lines are drawn to the two closest points, representing the path the player can travel down. I could have improved this implementation by including more than one point map to add more variation in the structure of the map created.
